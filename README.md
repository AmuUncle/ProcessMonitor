# Qt_ProcessMonitor

#### 介绍
本项目为windows下进程监控程序，支持实时获取指定进程的内存，cpu。可动态调节查询间隔，保存日志，
展示实时曲线图等。

#### 软件架构
- Qt 5.9
- Windows(x32, x64)
